Какие данные мы сохраняем?


  • ID серверов, к которым подключён бот
  • ID участников (временный бан, мьют, предупреждение)
  • Балансы, кейсы, префикс, модерация и т.д.
  • Количество выполненных команд
  • Выставленные счета
  • Ваши сообщения не сохраняются
Чтобы удалить свои данные обратиться на сервер поддержки. Поддержка

Для чего мы это делаем?

  • Для сохранения настроек бота
  • Для закрепления наказаний до момента их отмены
  • Для статистических целей
  • Для отслеживания информации
